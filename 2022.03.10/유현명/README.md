## Android FCM 

#### FCM이란?

사용자 디바이스에 설치된 앱 단위로 메세지를 전송할 수 있는 서비스이며 Firbase Cloud Messaging를 줄여서 FCM이라고 합니다. 즉, 안드로이드 푸쉬 알람 서비스 기능을 사용할 수 있도록 구글이 제공하는 무료 푸시 서버입니다.
